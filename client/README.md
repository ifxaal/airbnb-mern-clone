# StayEase Frontend

React frontend for the StayEase property booking platform.

## Scripts

- `npm start` - runs development server at `http://localhost:3000`
- `npm run build` - creates production build
- `npm test` - runs tests

## API Configuration

The app uses:

- `REACT_APP_API_URL` when provided
- `http://localhost:5000/api` in development
- `/api` in production
